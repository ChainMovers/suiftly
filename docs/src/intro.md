
# What is Suiftly?

Global CDN for Walrus

Will allow to swiftly load Blob in various manner:
(1) As a simple CDN link (e.g. https://suiftly.io/blob/myblob)
(2) Using Suiftly NPM package for blob loading and signature verification.
